### About This Project
This is my personal portfolio built from scratch to present myself to recruiters, companies, and the dev community.  
It features a modern, responsive single-page design with dark tech theme, smooth scrolling, hover effects, and clean code structure.

Key highlights:
- Fully responsive (mobile-first)
- Semantic HTML5 + CSS3 (Flexbox, Grid, custom variables, transitions)
- Vanilla JavaScript for smooth scroll and interactions
- Hosted on Vercel with automatic deploys from GitHub
- Clean, maintainable code with comments and organized sections

### Sections Included
- Hero / Introduction
- About Me (professional summary + location)
- Technical Skills (icons + grid)
- Projects (cards with descriptions, tags, GitHub & demo links)
- Professional Experience (timeline-style items)
- Education (ongoing degree)
- Contact Form (integrated with Formspree) + social links

### Technologies Used
- HTML5
- CSS3 (custom properties, media queries, animations)
- JavaScript (vanilla – smooth scroll, form handling)
- Font Awesome (for icons)
- Vercel (hosting & CI/CD)
- GitHub (version control)

### Project Journey
- Started in early 2025 as part of my self-improvement and job search preparation
- Built iteratively: first static HTML/CSS structure → added responsiveness → refined design
- Deployed on Vercel for instant previews and production URL
- Continuously updated with new projects, skills, and experience
